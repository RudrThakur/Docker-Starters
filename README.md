# Docker-Starters

Basic Template for Docker Beginners.

## Features

1. PHP
2. Nginx
3. MySQL
4. Composer
5. NPM
6. Artisan

## Instructions

Run using the command

```
docker-compose up -d --build

```
To Access Features through CLI

```
docker-compose run --rm <feature> <command>

```

To stop all features

```
docker-compose down

```
